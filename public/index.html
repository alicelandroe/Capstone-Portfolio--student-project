Projekt

Sign Up Formulär

I detta projekt byggde jag ett lyhört, mobilvänligt registreringsformulär med en mängd olika HTML-formulärinmatningstyper och attribut. Med hjälp av de medföljande mockupfilerna byggde jag en mobil- och skrivbordsversion av formuläret med media queries och en "mobil-först" -metod.

Foto Galleri

I detta projekt byggde jag ett interaktivt fotogalleri med JavaScript och jQuery. Överst på sidan finns ett sökområde där foton döljs och visas beroende på användarens inmatning. När användaren klickar på en miniatyrbild kommer fotot att visas i en lightbox.

Wheel of Success

I detta projekt skapade jag en webbläsarversion av "Wheel of Success", ett ordgissningsspel där spelare klickar på bokstäver från ett tangentbord på skärmen för att försöka gissa en slumpmässig fras.

Web App Dashboard

I det här projektet byggde jag en webbpanel komplett med JavaScript-driven diagram och grafer. Jag skapade alla HTML-, CSS- och JavaScript-funktioner för den här sidan "från scratch".

Style Guide

I detta projekt konverterade jag CSS till Sass genom att dela koden i flera Sass-delfiler. Jag letade också upp upprepade värden (längdenheter, färger etc.) i hela CSS och lagrar dem i Sass-variabler.

API Projekt

I detta projekt använde jag Random User Generator API (https://randomuser.me/) för att få information för 12 slumpmässiga "anställda" och använda den informationen för att bygga en prototyp för en Awesome Startup-anställdskatalog.
